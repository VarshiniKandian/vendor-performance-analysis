# Vendor Performance Analysis

## 📌 Overview
This project analyzes vendor performance data to identify trends, evaluate vendor efficiency, and support data-driven business decisions.

The analysis focuses on understanding sales distribution, vendor contribution, and performance patterns using data analytics and visualization tools.

---

## 🎯 Objectives
- Analyze vendor sales data  
- Identify top-performing and low-performing vendors  
- Understand sales distribution and trends  
- Support business decisions through data insights  

---

## 📊 Key Analysis Performed
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Vendor-wise sales analysis  
- Distribution analysis and trend identification  

---

## 📈 Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib & Seaborn  
- Power BI  

---

## 🔍 Key Insights
- A small number of vendors contribute significantly to total sales  
- Sales distribution varies across vendors, indicating performance gaps  
- Some vendors consistently outperform others, suggesting optimization opportunities
